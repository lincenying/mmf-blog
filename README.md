先安装 api server: https://github.com/lincenying/mmf-blog-api

```
// 安装cooking
npm install cooking-cli -g

// 安装依赖
npm install      or      yarn

// 生成静态文件
npm run build

// 开发模式
npm run dev

```

首页
http://localhost:8080
