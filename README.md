# mmf-blog vuejs 1.0版

本版本已不再更新, 请移步新版本: https://github.com/lincenying/mmf-blog-vue2

---

先安装 api server: https://github.com/lincenying/mmf-blog-api

```
// 安装cooking
npm install cooking-cli -g

// 安装依赖
npm install      or      yarn

// 生成静态文件
npm run build

// 开发模式
npm run dev

```

首页
http://localhost:8080

登录
http://localhost:8080/login.html
